# Start Development Servers

Starts the Express backend and the Vite React frontend in parallel.

## How to Run

**Option 1 — Ask the AI:** Type `/dev` in chat

**Option 2 — Run manually:** Open **two** terminals:

**Terminal 1 (Backend):**
```bash
cd D:\Portfolio\3rdYear\CapstoneSystem\Capstone-Website\server
npm run dev
```

**Terminal 2 (Frontend):**
```bash
cd D:\Portfolio\3rdYear\CapstoneSystem\Capstone-Website\web
npm run dev
```

## Expected Output
- Backend: `🚀 LILYCREST DORMITORY MANAGEMENT SYSTEM started` on port `5000`
- Frontend: `Local: http://localhost:3000`
