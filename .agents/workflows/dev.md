# Start Development Servers

This workflow starts the Express backend and the Vite React frontend in parallel.

// turbo-all

1. Start the backend server:
```bash
cd D:\Portfolio\3rdYear\CapstoneSystem\Capstone-Website\server && npm run dev
```

2. Start the frontend dev server:
```bash
cd D:\Portfolio\3rdYear\CapstoneSystem\Capstone-Website\web && npm run dev
```

3. Confirm both servers are running by checking the terminal output for:
   - Backend: `Server running on port ...`
   - Frontend: `Local: http://localhost:...`
