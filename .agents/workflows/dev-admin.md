# Start Admin Panel Dev Servers

Starts the Express backend and the admin Vite frontend build.

## How to Run

**Option 1 — Ask the AI:** Type `/dev-admin` in chat

**Option 2 — Run manually:** Open **two** terminals:

**Terminal 1 (Backend):**
```bash
cd D:\Portfolio\3rdYear\CapstoneSystem\Capstone-Website\server
npm run dev
```

**Terminal 2 (Admin Frontend):**
```bash
cd D:\Portfolio\3rdYear\CapstoneSystem\Capstone-Website\web
npm run dev:admin
```

## Expected Output
- Backend: `🚀 LILYCREST DORMITORY MANAGEMENT SYSTEM started` on port `5000`
- Admin Frontend: `Local: http://localhost:3001`
