# Start Admin Panel Dev Servers

This workflow starts the Express backend and the admin Vite frontend build.

// turbo-all

1. Start the backend server:
```bash
cd D:\Portfolio\3rdYear\CapstoneSystem\Capstone-Website\server && npm run dev
```

2. Start the admin frontend dev server:
```bash
cd D:\Portfolio\3rdYear\CapstoneSystem\Capstone-Website\web && npm run dev:admin
```

3. Confirm both servers are running by checking the terminal output for:
   - Backend: `Server running on port ...`
   - Admin Frontend: `Local: http://localhost:...`
